fstats::t_test_paired Interface Reference

Computes the 2-tailed Student's T-Test for two paired data sets. More...

Detailed Description

Computes the 2-tailed Student's T-Test for two paired data sets.

Syntax
subroutine t_test_paired(real(real64) x1(:), real(real64) x2(:), real(real64) stat, real(real64) p, real(real64) dof, optional class(errors) err)
subroutine t_test_paired(real(real32) x1(:), real(real32) x2(:), real(real32) stat, real(real32) p, real(real32) dof, optional class(errors) err)
Parameters
[in]x1An N-element array containing the first data set.
[in]x2An N-element array containing the second data set.
[out]statThe Student-'s T-Test statistic.
[out]pThe probability value that the two samples are likely to have come from the same two underlying populations that have the same mean.
[out]dofThe degrees of freedom.
[in,out]errA mechanism for communicating errors and warnings to the caller. Possible warning and error codes are as follows.
  • FS_NO_ERROR: No errors encountered.
  • FS_ARRAY_SIZE_ERROR: Occurs if x1 and x2 are not the same length.
