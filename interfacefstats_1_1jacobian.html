Detailed Description

Computes the Jacobian matrix for a nonlinear regression problem.

Syntax
subroutine jacobian(pointer procedure(regression_function) fun, real(real64) xdata(:), real(real64) params(:), real(real64) jac(:,:), logical stop, optional real(real64) f0(:), optional real(real64) f1(:), optional real(real64) step, optional class(errors) err)
Parameters
[in]funA pointer to the regression_function to evaluate.
[in]xdataThe M-element array containing x-coordinate data.
[in]paramsThe N-element array containing the model parameters.
[out]jacThe M-by-N matrix where the Jacobian will be written.
[out]stopA value that the user can set in fun forcing the evaluation process to stop prior to completion.
[in]f0An optional M-element array containing the model values using the current parameters as defined in m. This input can be used to prevent the routine from performing a function evaluation at the model parameter state defined in params.
[out]f1An optional M-element workspace array used for function evaluations.
[in]stepThe differentiation step size. The default is the square root of machine precision.
[in,out]errA mechanism for communicating errors and warnings to the caller. Possible warning and error codes are as follows.
  • FS_NO_ERROR: No errors encountered.
  • FS_ARRAY_SIZE_ERROR: Occurs if any of the arrays are not properly sized.
  • FS_OUT_OF_MEMORY_ERROR: Occurs if a memory error is encountered.
