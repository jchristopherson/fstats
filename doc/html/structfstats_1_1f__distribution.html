Remarks
The CDF for a F distribution is given as \( F(x) = I_{d_1 x/(d_1 x + d_2)} \left( \frac{d_1}{2}, \frac{d_2}{2} \right) \) where \( I \) is the regularized incomplete beta function.

procedure, public fstats::f_distribution::pdf

Computes the probability density function.

Syntax
real(real64) function pdf(class(f_distribution) this, real(real64 )x)
Parameters
[in]thisThe f_distribution object.
[in]xThe value at which to evaluate the function.
Returns
The value of the function.
Remarks
The PDF for a F distribution is given as \( f(x) = \sqrt{ \frac{ (d_1 x)^{d_1} d_{2}^{d_2} }{ (d_1 x + d_2)^{d_1 + d_2} } } \frac{1}{x \beta \left( \frac{d_1}{2}, \frac{d_2}{2} \right) } \) where \( \beta \) is the beta function.
