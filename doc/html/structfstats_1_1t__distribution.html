Returns
The value of the function.
Remarks
The CDF for Student's T-Distribution is given as \( F(t) = \int_{-\infty}^{t} f(u) \,du = 1 - \frac{1}{2} I_{x(t)} \left( \frac{\nu}{2}, \frac{1}{2} \right) \) where \( x(t) = \frac{\nu}{\nu + t^2} \), \( I \) is the regularized incomplete beta function, and \( \nu \) is the number of degrees of freedom. This formula is valid for \( t > 0 \). For values of \( t < 0 \), the symmetry of the function can be exploited.

Remarks
The PDF for Student's T-Distribution is given as \( f(t) = \frac{ \Gamma \left( \frac{\nu + 1}{2} \right) }{ \sqrt{\nu \pi} \Gamma \left( \frac{\nu}{2} \right) } \left( 1 + \frac{t^2}{\nu} \right)^{-(\nu + 1) / 2} \) where \( \nu \) is the number of degrees of freedom.
