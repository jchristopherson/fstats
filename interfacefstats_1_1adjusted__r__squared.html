Detailed Description

Computes the adjusted R-squared value for a data set.

Syntax
Parameters
[in]pThe number of model parameters.
[in]xAn N-element array containing the dependent variables from the data set.
[in]xmAn N-element array containing the corresponding modeled values.
[in,out]errA mechanism for communicating errors and warnings to the caller. Possible warning and error codes are as follows.
  • FS_NO_ERROR: No errors encountered.
  • FS_ARRAY_SIZE_ERROR: Occurs if x and xm are not the same size.
Returns
The result.
The adjusted R-squared provides a mechanism for tempering the effects of extra explanatory variables on the traditional R-squared calculation. It is computed by noting the sample size \( n \) and the number of variables \( p \): \( \bar{R}^2 = 1 - \left( 1 - R^2 \right) \frac{n - 1}{n - p} \).
